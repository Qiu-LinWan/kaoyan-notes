手机强推送

支持云端模式

这里有两种模式。 本地模式适合先快速试用；云端模式适合你现在这个需求：电脑关机也能照样推送到手机。

云端模式（推荐）

下载云端配置文件后，把这个项目上传到 GitHub， 再在仓库里配置 PUSHPLUS_TOKEN Secret。 GitHub Actions 会每 5 分钟检查一次，到你的提醒时间后自动发送。

GitHub Secret 名称

PUSHPLUS_TOKEN

GitHub Variable 名称

KAOYAN_PROFILE_JSON

本地模式

这部分会导出你的推送配置，并配套两个本地脚本： 一个负责每天生成并发送今日计划，一个负责注册 Windows 定时任务。 前提是你的电脑在提醒时间保持开机并联网。

安装命令

测试命令

1. 先在 PushPlus 获取你的 token，然后填到上面的输入框。

2. 云端模式：复制云端 JSON，上传项目到 GitHub 仓库，并把 token 存成 GitHub Secret：PUSHPLUS_TOKEN，再把 JSON 存成 GitHub Variable：KAOYAN_PROFILE_JSON。

3. 本地模式：下载 push-config.json，运行安装命令后，Windows 会在你设定的时间自动发送当天计划到手机。
